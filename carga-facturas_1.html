CÓMO ARMAR EL ESCENARIO EN MAKE
1Webhooks → Custom webhook — creá uno nuevo, copiá la URL y pegala arriba
2HTTP → Make a request con estos datos:
· URL: https://api.anthropic.com/v1/messages
· Método: POST
· Body type: Raw → application/json
3Headers del módulo HTTP:
· x-api-key → tu API key de Anthropic
· anthropic-version → 2023-06-01
· content-type → application/json
4Body del módulo HTTP (copiá esto):
{ "model": "claude-opus-4-5", "max_tokens": 1000, "messages": [{ "role": "user", "content": [ { "type": "image", "source": { "type": "base64", "media_type": "image/jpeg", "data": "{{1.imagen}}" } }, { "type": "text", "text": "Extraé los datos de esta factura argentina en JSON puro sin markdown:\n{\"fecha\":\"DD/MM/YYYY\",\"cuit\":\"solo numeros\",\"proveedor\":\"nombre\",\"concepto\":\"descripcion o null\",\"tipoComprobante\":\"Fc A/B/C\",\"nroComprobante\":\"XXXX-XXXXXXXX\",\"neto\":numero,\"iva21\":numero,\"iva105\":numero,\"otros\":numero,\"percIva\":numero,\"percIibbBa\":numero,\"percIibbCaba\":numero,\"total\":numero,\"quien\":null}" } ] }] }
5Tools → JSON → Parse JSON — parseá {{2.data.content[].text}}
6Microsoft Excel 365 → Add a Worksheet Row
Seleccioná tu archivo → hoja Compras 2026
Mapeá: {{3.fecha}}, {{3.cuit}}, {{3.proveedor}}, etc.
